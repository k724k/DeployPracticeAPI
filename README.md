# 🏷️ DeployPracticeAPI 
Spring Boot CRUD 백엔드를 GitHub Actions, Docker, AWS(Route53, ELB, RDS, S3) 기반으로 CI/CD 자동 배포 파이프라인까지 구축한 실습 프로젝트입니다.

## ✅ Overview
- 기본 CRUD 생성 및 데이터베이스 연결  
- GitHub 및 GitHub Desktop을 활용한 소스 코드 버전 관리  
- GitHub Actions를 통한 CI/CD 파이프라인 구축 및 자동화  
- Dockerfile 작성과 Docker 이미지 빌드에 의한 컨테이너화 및 배포  
- AWS 인프라 활용
  - Route53을 통한 도메인 네임서버 연결 및 관리  
  - ELB(Application Load Balancer) 설정과 HTTPS 인증서 적용  
  - RDS(MySQL) 인스턴스 생성 및 Spring Boot 서버와 데이터베이스 연동  
  - AWS S3를 이용한 이미지 업로드 기능 설계 및 구현

## ✅ Features
- 안정적인 CRUD 기능 구현 및 데이터베이스 연동  
- 효과적인 코드 관리 및 협업 환경 구축  
- CI/CD 파이프라인 자동화를 통한 빠르고 안정적인 배포 지원  
- Docker 기반 컨테이너화로 이식성과 배포 편의성 강화  
- AWS 인프라를 활용한 서비스 안정성과 확장성 확보  
- 클라우드 스토리지 연동 이미지 업로드 기능 포함
